# react-native-zendesk-sdk

React Native ZndeskSDK Wrapper

## Installation

```sh
npm install react-native-zendesk-sdk
```

## Usage

```js
import ZendeskSdk from "react-native-zendesk-sdk";

// ...

const result = await ZendeskSdk.multiply(3, 7);
```
